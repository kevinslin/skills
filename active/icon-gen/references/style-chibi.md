# Style: chibi

Use this style when the user does not name a style explicitly.

## Intent

Create a cute, compact mascot-like icon with oversized facial features, rounded forms, and a friendly expression. The result should feel polished enough for branding, not like a random sticker pack.

## Prompt additions

Use these ideas to augment the user's request without inventing a new subject:

- Style/medium: chibi mascot icon; flat illustration; vector-friendly; crisp outlines; minimal shading
- Composition/framing: single centered icon with generous padding for the general icon; readable at small sizes
- Lighting/mood: bright, playful, polished
- Materials/textures: flat colors, crisp edges, simple geometric forms
- Constraints: transparent background; strong silhouette; balanced negative space; no watermark; no mockup; no surrounding scene
- Avoid: photorealism, busy background, 3D render, stock mascot vibe, extra props, text-heavy design

## Inline-icon bias

For icons that will also have an inline variant:

- Favor a left-to-right or centered silhouette that still reads after tight cropping.
- Keep fine details secondary to the main face and outline.
- Avoid large empty transparent margins at the bottom.
